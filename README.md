# CovidSense
**桂城中学疫情数据**

新增病例以核酸检测或抗原自测呈阳性为判断依据，所有数据仅供参考。

## 数据结构

所有数据存储在 `data.json` 中。

```
[
    {
        "date": [2023, 1, 1] // 年月日
        "data": [
            [0, 1, 2, 3], // 高一，2-5层
            [0, 1, 2, 3, 4], // 高二，楼层不详的在最后补充
            [0, 1, 2, 3], // 高三
            [1] // 教师
        ]
    }
]
```
